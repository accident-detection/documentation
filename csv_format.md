CSV format
=============

The CSV record is formated as follows:

```time```;```date```;```GPS long```;```GPS lat```;```GPS alt```;```GPS message code```

--------------------

JSON record is formated as follows:

```
  data = {
    time,
    date,
    GPSlong,
    GPSlat,
    GPSalt,
    GPScode,
  }  
```
